Army Center Targeting: Current Execution Flow

How AiOrchestrator::order() uses MilitaryBrain center-pair pressures and unit positions to issue army commands directly toward selected influence-map centers.

Two Decisions

POSITIONALWhich objective wins for this unit?

Every army unit is evaluated independently. Its distance to the current influence-map centers determines how strongly it receives each directional MilitaryBrain pressure. The highest score selects the unit's objective bucket.

DIRECT CENTERWhere does the selected bucket go?

The bucket goes directly to the center it selected. There is no second brain, weighted map aggregation, candidate list, or shared areas[0] target.

Fog of war is intentionally ignored: enemy and battle centers remain valid targets even when stale or hidden. If a center does not exist, units assigned to it enter the hold path.

Data Flow

Worked Example: Three Units

Assume the current center snapshots are own economy (100, 100), own building (110, 100), and enemy army (800, 750). The MilitaryBrain pressures cause the three units to select different centers. Each unit moves to that selected center, not to a common candidate cell.

Commands and Edge Cases

FARGo to the center

If squared distance to the selected center exceeds 30 * 30, the unit receives GO(targetCenter). Its calculated pressure becomes the command priority.

CLOSEAttack locally

When within 30 units of the selected center, the unit does not queue a new move. It attacks the closest nearby enemy if one exists.

LOW PRESSUREHold

If the best calculated score is <= 0.1, the unit enters the hold group. Free hold units can still attack a nearby enemy.

MISSING CENTERHold

If the selected influence center is unavailable, every unit in that bucket enters the hold group. There is no alternate heatmap target or center fallback.

Command priority guard

A new command is only queued when its pressure exceeds the unit's current command priority. At the start of each order(), priority decreases by 0.25. This prevents weaker new objectives from replacing a stronger current command.
