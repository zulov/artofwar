Output Flow — How Outputs Cascade
MasterBrain outputs 9 urgencies that feed into sub-brains. Sub-brains produce requests for the WantList or direct game orders. All outputs are tanh [-1, +1].
1. MasterBrain — 9 Urgency Outputs
Pure strategic signals, not commands. Each urgency tells a sub-brain "how much to care" about its domain. The same attackUrgency drives UnitBrain to pick offensive units AND MilitaryBrain to attack rather than defend.
2. UnitBrain — 24-Dimensional Profile Matching
The NN outputs an abstract "ideal unit" description. This is distance-matched against every real unit in the nation's roster. The NN learns unit characteristics, not unit IDs — enabling transfer across nations with different rosters.

Output Type Legend

URGENCY — Strategic priority signal. Fed as input to sub-brains and/or used as WantList request priority.

PRIORITY — Relative preference among alternatives. Sorted or argmax'd to pick between options.

PROFILE — Abstract feature vector. Distance-matched against real entity metrics.

DECISION — Direct action trigger. Thresholded or compared to trigger specific game actions.

UNUSED — Defined but not yet consumed. Planned for future implementation.
